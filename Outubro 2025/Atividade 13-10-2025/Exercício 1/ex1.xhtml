O Último Raio de Algoritmo

Quando o sol da era digital começou a se pôr, restou apenas uma linha de código pulsando na imensidão dos servidores abandonados. Diziam que aquele algoritmo era o último resquício da humanidade - uma tentativa desesperada de registrar sentimentos em linguagem binária. Entre zeros e uns, batia o eco distante de um coração que se recusava a desligar.

No silêncio metálico das máquinas adormecidas, uma jovem engenheira encontra esse fragmento perdido. À medida que o decifra, percebe que cada comando guarda lembranças, promessas e sonhos de um mundo que esqueceu o que era sentir. E quando o algoritmo enfim desperta, o céu noturno se ilumina com um último raio - não de luz, mas de memória.


        public class UltimoRaioDeAlgoritmo {

        public static void main(String[] args) {
            SistemaHumanidade sistema = new SistemaHumanidade();
            sistema.iniciar();
        }
    }

    class SistemaHumanidade {
        private boolean humanidadePerdida = true;
        private String[] memoria = {
            "lembrança", 
            "promessa", 
            "sonho", 
            "silêncio"
        };

        public void iniciar() {
            System.out.println(">> Iniciando o último raio de algoritmo...");
            for (String dado : memoria) {
                processar(dado);
            }
            despertar();
        }

        private void processar(String dado) {
            System.out.println("Processando " + dado + "...");
            pausar(1000);
        }

        private void despertar() {
            if (humanidadePerdida) {
                System.out.println("\n// Decifrando emoções esquecidas...");
                pausar(1500);
                System.out.println("if (coração.bate()) humanidadePerdida = false;");
                humanidadePerdida = false;
            }

            if (!humanidadePerdida) {
                System.out.println("\n>> O algoritmo despertou.");
                System.out.println(">> Um último raio — não de luz, mas de memória — atravessa o céu digital.");
            }
        }

        private void pausar(int ms) {
            try {
                Thread.sleep(ms);
            } catch (InterruptedException e) {
                System.err.println("Erro ao tentar sonhar...");
            }
        }
    }
